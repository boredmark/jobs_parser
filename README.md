# Jobs Parser

## Features

- Collecting 50 last posted jobs from https://www.freelancer.com
- Saving results to csv file (Title, price, link) in root directory

## Installation

```sh
cd <path>
git clone
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Run

```sh
python3 main.py
```
